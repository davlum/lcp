# LessCP

Kind of like [less][1], but then adds the highlighted results to the clipboard.

Inspired by/built upon https://www.flenker.blog/hecto/.

## Installation

### Binaries

Download one of the binary releases.

### From source

Install from source with `cargo`.

```bash
cargo install --path .
```
